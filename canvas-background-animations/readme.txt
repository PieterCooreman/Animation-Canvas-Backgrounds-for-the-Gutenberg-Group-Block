Canvas Background Animations adds beautiful canvas background animations to Gutenberg Group blocks

== Description ==

When selecting a Group-block, you can add various JavaScript-driven Canvas background animations. You can nest groups and have multiple animations combined.

== Installation ==

= Installation from within WordPress =

1. Visit **Plugins > Add New**.
2. Search for **Canvas Background Animations**.
3. Install and activate the Canvas Background Animations plugin.

= Manual installation =

1. Upload the entire `Canvas Background Animations` folder to the `/wp-content/plugins/` directory.
2. Visit **Plugins**.
3. Activate the Canvas Background Animations plugin.
